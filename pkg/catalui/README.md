Catalui
=======


Presentation
------------

*Catalui* is an Astro package for generating the frontend of *Catalos*.


Dev
---

```bash
git clone https://github.com/charlyoleg2/catalos
cd catalos
npm i
npm -w catalui run ci
npm -w catalui run preview
npm -w catalui run clean
```
